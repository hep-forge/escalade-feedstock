Feedstock for [escalade](https://gitlab.cern.ch/escalade/framework) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Escalade Framework: File Processing Middleware on HPC computers
